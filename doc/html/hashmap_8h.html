void ds_hashmap_sort_by ( struct DSHashMap *  hash,
int32_t(compare)(void *, void *)   
)

Sorts the keys using the provided compare function. The compare function is defined as follows: compare(k1, k2) < 0 when k1 < k2 compare(k1, k2) = 0 when k1 = k2 compare(k1, k2) > 0 when k1 > k2

NOTE: The comparison doesn't have to be on the keys directly. One could use ds_get and compare those values.
